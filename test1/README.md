# AI Web App Starter: 미디어 서비스 레이아웃

## 프로젝트 개요
- 본 프로젝트는 프론트엔드 기초 학습을 위해 `HTML`과 `CSS`만 사용하여 구현한 미디어 콘텐츠 서비스 메인 페이지 뼈대입니다.
- 시맨틱 태그(`header`, `main`, `footer` 등)를 활용해 접근성과 문서 구조를 함께 고려했습니다.
- 화면 구성의 핵심 골격을 먼저 만드는 것을 목표로 하며, 이후 반응형/컴포넌트 확장 학습에 이어질 수 있도록 설계했습니다.

## 주요 특징
- 시맨틱 마크업 준수 (`header`, `main`, `section`, `article`, `footer` 활용)
- 의미 있는 클래스명 사용으로 가독성 및 유지보수성 고려
- 추후 카드 확장 및 반응형 적용이 가능하도록 구조 중심의 CSS 작성

## 기술 스택
- HTML5
- CSS3

## 개발 로그
### 사용 도구
- Cursor + Claude API

### 요청 내용
- 미디어 서비스 메인 화면 Skeleton 생성

### 사용 프롬프트 요약 (1~4)
1. `index.html`, `styles.css` 기반의 아주 간단한 웹페이지 초기 구조 생성
2. 브라우저에서 바로 실행 가능한 형태로 확인
3. 헤더/배너/카드/푸터를 갖춘 미디어 콘텐츠 서비스 메인 페이지 뼈대 구성
4. 기존 클래스명을 유지하면서 카드 섹션 구조를 더 명확하게 정리

### 결과
- 헤더/대표 배너/카드 섹션(3개 카드)/푸터 구조 생성 성공

### 보완점
- 카드 섹션 구조를 더 명확히 수정 예정
